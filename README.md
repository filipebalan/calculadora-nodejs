🧮 Calculadora de Terminal com Node.js

Meu primeiro projeto prático utilizando JavaScript e o ambiente Node.js. 

O objetivo deste projeto foi construir a lógica fundamental de uma calculadora operando diretamente no terminal, focando em entender os conceitos essenciais do back-end antes de partir para frameworks.

🚀 O que eu aprendi
* Configuração e uso do módulo nativo `readline` do Node.js.
* Controle de fluxo assíncrono com Funções de Retorno (Callbacks).
* Estrutura de decisão lógica usando `switch / case`.
* Captura de entrada de dados do usuário via terminal (Standard Input) e conversão de Tipos (Strings para Numbers).

🛠️ Como rodar o projeto
1. Certifique-se de ter o Node.js instalado na sua máquina.
2. Baixe o arquivo `calculadora.js`.
3. Abra o seu terminal na pasta do arquivo e rode o comando:
   ```bash
   node calculadora.js
